How are the internet and Web Different?

The Web is a network of websites that are hosted on the internet. It consists of websites and web pages. The internet is a global network of interconnected computers and servers

ASCII and Unicode

ASCII and Unicode are both character encoding standards. ASCII is a short for American Standard for information Interchange and it uses a maximum of eight bits to represent 256 characters. Unicode is an encoding standard that represents over one million characters

UTF-8

It is a feature that converts Unicode characters into a series of 8-bit bytes

URL

It is the web address of a web page

Domain Name

It is a unique name that identifies website IP's address on the internet

HTTP

It is the abbreviated form for HyperText Trasmission Protocol. Its a set of rules that allow web clients and servers to communicate with each other.

HTTPS

Its stands for HyperText Trasmission Protocol Secure. It is the secured version of HTTP

IP Address

It is an assigned number to devices that connect or identifies them to or on the internet

DNS

It stands for Domain Name System. It is a system that translates Domain names into IP addresses

W3C Recommendations

These are official guidelines and standards that define web technologies like HTML, CSS, and JavaScript should interact and work

XML

XHTML

HTML
